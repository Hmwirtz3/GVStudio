Gravitas Studio is a lightweight, purpose-built scene editor designed for creating and exporting game-ready data for the Gravitas Runtime, which has its current target as the Sony PSP. It provides a streamlined workflow for assembling scenes, configuring logic , and packaging assets into an efficient, chunk-based binary format tailored for low-level runtimes like the PSP.
